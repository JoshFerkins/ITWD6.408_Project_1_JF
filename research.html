Describe basic principles of copyright, 4 types of CC licences and Fair Use with image as illustration

The expression of an idea in an original work is legally protected by copyright as soon as it becomes 'recorded in material form'.
🅯Attribution : User is allowed to distribute, remix, adapt, or build as long as credit is given to the original work.
🅯🄎Attribution-ShareALike : Same restrictions as attribution, users must use the same licenses as original.
🅯⊜Attribution-NoDerivs : Same restrictions as attribution, users may reuse work, without sharing in the adapted form.

Search Engine
Optimisation

Describe 5 ways to improve the visibility of your website on search engine.

Keywords : By using the <meta> tags, you can specify keywords for you website to use in web searchs.
Optimising Pages : By linking your pages with keywords, url links, and meta descriptions, google can easily crawl and index your pages.
Verification : By getting your website verified by the specific browsers, it will be prioritised for search results.
SEO Audit : You may prepare your website with an SEO audit to test the website performance including errors, links or redirects.
Site Map : By submitting a site map of your website to webmaster tools, the search engine will be able to crawl and index you pages with ease.

Web Security

Describe 5 common cyber attacks (threats) and preventive measures against those attacks

DoS : This is a Denial of Service attack which floods the server with useless information slowing it. Reinforce security for internet-facing devices
XSS : Cross-site Scripting is done by injecting scripts into websites with malicious intent. Filtering information inputs
SQLI : SQL Injection, injecting SQL statements into servers or databases to retrieve or modify information. Prepared statements and input validation
Path Traversal : Allows access to application files. Avoid passing user-supplied input to filesystem APIs
Local File Inclusion : Forces the application to run a specific file. Use secure databases, reduce incorporating local files
https://www.trustnetinc.com/web-application-attacks/

DoS : Reinforce security for internet-facing devices.
XSS : Cross-site Scripting, filtering inputs.
SQLI : SQL Injection, prepared statements and verified inputs.
Path Traversal : Avoid passing user-supplied input to filesystem APIs.
Local File Inclusion : Secure databases, reduce local files.
